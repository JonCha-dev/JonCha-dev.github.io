# Projects: 
### ------------------------------------------------------------------------------------------

[ISA Project](https://github.com/vmvazquez/CSE141L-Processor) (Currently a private repo)

- A dedicated Instruction Set Architecture designed in order to accomplish three tasks: generate Hamming Code bits, perform one-bit error correction and two-bit error detection, and count occurences of a specified bit pattern in memory.


[SurfStore](https://github.com/JonCha-dev/SurfStore/tree/main/SurfStore-j3chang-main) 

- A cloud-based file storage system implemented through golang. Allows multiple clients to sync files by pushing/pulling version changes to/from a remote server, comparing the versions between the local files and remote files on the server. Also supports deleting files.

[UPick](https://github.com/leejoongyin/FoodForagersUPick)

- An application built on JavaScript and React that helps users decide where they should go to eat based on their dietary preferences. Supports both single-user and group functionality. 

[Nachos](https://github.com/JonCha-dev/Nachos)

- (Not Another Completely Heuristic Operating System) An Instructional OS with various modules implementing the functionality of a basic Operating System. For the project, various functions related to system calls, multithreading, and concurrency were implemented. 

[Data Science Project](https://github.com/JonCha-dev/DataScienceProj/blob/main/FinalProject_group016.ipynb)

- Utilizing Python, an observation was done on the relationship between exonerations and race in the United States. The data was pulled from a public dataset and cleaned for analysis. Then, the distributions for the measured variables were visualized and analyzed. Finally, inferential analysis was done on the resulting data to draw a more accurate conclusion.
